AI Real-Time Voice Translation Assistant User Manual

1. Basic Functions
o	Core capabilities : Real-time translation and bilingual display based on voice input. Supports mutual translation between 10 source languages, including English, Russian, and French, and 4 target languages, with automatic text recording.
o	Based on cloud services : When connected to the internet, the domestic real-time speech recognition and translation model (gummy-realtime-v1) under Alibaba Cloud services can be used to a certain extent in scenarios with complex accents or background noise.
o	Multi-scenario adaptability : It can be used for multiple scenarios such as simultaneous text interpretation in public meetings, live streaming, local video playback and transcription, and foreign language learning .

2. Supported languages

2.1 Recognizable Languages (Audio Input Language)
o	Chinese ( zh ), English (en), Japanese ( ja ), Korean ( ko )
o	Cantonese ( yue ), German (de), French ( fr ), Russian ( ru )
o	Italian (it), Spanish ( es )

2.2 Language Mapping
o	Bidirectional translation : Chinese ⇌ English, Chinese ⇌ Japanese, Chinese ⇌ Korean, English ⇌ Japanese
English ⇌ Korean
o	One-way translation : Cantonese/German/French/Russian/Italian/Spanish →
Chinese/English
